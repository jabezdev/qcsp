# QCSP Volunteer Matrix

This is a volunteer management system for QCSP. It allows managing volunteers, program teams, assignments, and committee structures.

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- @dnd-kit (for drag and drop functionality)
- Express (Backend Server)

## Getting Started

### Prerequisites

- Node.js & npm

### Installation

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```

### Development

To run the frontend:
```sh
npm run dev
```

To run the backend server:
```sh
npm run server
```

### Production Build

```sh
npm run build
npm start
```
